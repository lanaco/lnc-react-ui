# Page Layout

<SyntaxHighlighter language="jsx" style={nord}>
  {
    'import { PageLayout, Sidebar, Header, Footer, Content } from "@lanaco/lnc-react-ui";'
  }
</SyntaxHighlighter>

Used to structure the layout of your application, without styles. Considering that, you are in charge of styling all the pieces,
or you can use the predefined components.

## Component overview

Every component is only a semantic container for your content, which is laid out based on flex.

- `PageLayout` is the top-level wrapper in which you can nest `Header` `Footer` `Content` `Sidebar` or `PageLayout` itself.
- `Header` is intended to be used at the top of the `PageLayout`.
- `Sidebar` is intented to be used at either right or left side of `PageLayout`, and only has basic collapse function.
- `Content` is intented as the central piece of `PageLayout`.
- `Footer` is intended to be used at the bottom of the `PageLayout`.

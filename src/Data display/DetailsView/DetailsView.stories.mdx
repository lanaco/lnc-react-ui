#### `DetailsView` is inteded for quick implementation of simple detail views. It utilizes `FlexGrid` to create layout.

## fields

`fields` is an array that accepts object of type `{label: <string>, accessor: <string>, ...}`, where the rest of this object props (aside from `label` and `accessor`) are props of component <u>[FlexGridItem](?path=/docs/layout-flex-grid--default-story).</u>
